物件導向概念

Posted on 2019-04-11 | In OOP |

基本概念 (類別、物件)

類別

為一個物件的抽象概念，類別的定義包含物件的屬性，或是操作資料的函式。

比如說:現在有一個class car，這個class可以記錄汽車的品牌、馬力、型號等。

public class car{

public string brand;

public string type;

public int hp;

public  GetCarInformation(){

    return brand+type+hp;

}

}

物件

實作class，class的實體。

car BMW = new car(“BMW”,”330i”,”250p”); //宣告一個BMW的car之實體

Encapsulation(封裝)

為OOP重要概念之一，一個外部程式碼若要取得class內的private data就只能透過class定義的function去操作，保護程式碼，使用這個class的程式員也不需要知道他怎麼操作的。
